# Wythoff-diagrams
About
-------
LaTeX "package" to quickly draw Wythoff diagrams like to ones in Coxeter, H. S. M. "Wythoff's construction for uniform polytopes." Proceedings of the London Mathematical Society 2.1 (1935): 327-339.
